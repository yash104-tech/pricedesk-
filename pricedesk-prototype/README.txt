=========================================
PriceDesk Prototype Instructions
=========================================

Because modern web browsers prevent JavaScript files from loading directly via double-clicking "index.html" (due to local file protocol security restrictions), we have included a zero-install launcher:

1. Double-click the file named "run-prototype.bat" in this folder.
2. A window will open, and the prototype will automatically launch in your web browser at:
   http://localhost:8080/
3. You can click the "Quick Demo Access" buttons at the bottom of the login screen to sign in as any role.
4. When finished, simply close the console window to stop the server.
